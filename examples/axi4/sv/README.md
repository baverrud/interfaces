# AXI4 — SystemVerilog Examples

| File | Description |
|------|-------------|
| `sim.bat` / `synth.bat` | Convenience wrappers (call `engine.py`) |
| `sources.f` | Source manifest |
| `rtl/` | Design sources (master, slave, write/read split, tops) |
| `tb/` | Self-checking testbenches |
| `sim/` | Simulation output (gitignored) |
| `viv/` | Vivado project output (gitignored) |
| `constrs/` | XDC constraint files |

See `../README.md` for design descriptions and usage.
