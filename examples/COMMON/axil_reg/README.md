# AXI4-Lite Register File

Decoupled AXI4-Lite register file with separate write and read
interfaces.  Used by the `axilite` example designs.

| File | Language | Description |
|------|----------|-------------|
| `axil_reg.sv` | SystemVerilog | SV implementation with `axilite_if` modports |
| `axil_reg.vhd` | VHDL | VHDL implementation with `axilite_pkg` mode views |
